UserInfo component displays user information and handles user data editing.

This component renders user information, including the title (e.g., 'Username'), an input field for editing the data, and optional validation error messages. It allows editing of user data and updates the editedUser state when changes are made. If nestedProp is provided, it updates the nested property; otherwise, it updates the property directly.
